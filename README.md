## Cloud-Based Data Processing (IN2386)
* URL Shortener

## Build server binary
* Can be found in the following directory: build/bin

```
$ make build_server
```

## Build client binary
* Can be found in the following directory: build/bin

```
$ make build_client
```

## Build server docker image

```
$ make release_server
```

## Build client docker image

```
$ make release_client
```

## Tests

```
$ make test
```
